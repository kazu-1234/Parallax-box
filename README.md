# Parallax World

**Version:** v7.1.0

## 概要
Parallax Worldは、Webブラウザ上で動作するインタラクティブな3D空間アプリケーションです。
Three.jsとMediaPipeを活用し、デバイスのジャイロセンサー、タッチ操作、あるいはWebカメラによる顔認識（Face Tracking）を用いて、奥行きのある視差効果（Parallax）を体験できます。

v7.1.0では「GAME」モード（Face Dodge）にアップデートが加わり、スコアアップアイテムが登場しました。

## 特徴 (Features)

### 3つのシーン
1.  **📦 BOX**: 
    *   3D空間の奥行きと視差効果をシンプルに楽しむモードです。
2.  **🤖 DANCE**: 
    *   3Dキャラクター（ロボット）のダンスパフォーマンスを鑑賞、操作できるモードです。
3.  **🎮 GAME (Face Dodge)**: 
    *   迫りくる障害物を回避し、ハイスコアを目指すアクションゲームです。
    *   **New**: 黄色のクリスタル（ボーナスアイテム）を取るとスコアが大幅にアップします。

### 3つの操作モード
*   **GYRO (ジャイロ)**: スマートフォンやタブレットの傾きに合わせて視点が移動します。
*   **TOUCH (タッチ)**: 画面のドラッグやスワイプ、マウス操作で視点を移動します。
*   **FACE (顔認識)**: Webカメラでユーザーの顔の位置を認識し、顔の動きに合わせてリアルタイムに視点が移動します（※カメラ権限が必要です）。

## 操作方法 (Controls)

### PCをご利用の場合
*   **基本操作**: マウスドラッグ、またはキーボード操作
*   **キーボード**: `W` `A` `S` `D` キー、または矢印キー `↑` `↓` `←` `→`

### スマートフォン・タブレットをご利用の場合
*   **基本操作**: 端末の傾き（GYROモード時）、または画面スワイプ
*   **注意**: iOS端末などでジャイロセンサーを使用する場合、初回に許可が必要な場合があります。

## 使用技術
*   HTML5 / CSS3
*   Three.js (r128)
*   MediaPipe Face Mesh
